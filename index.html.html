Fault recommendations
⚡ Short circuit
URGENT
1. Switch OFF system immediately
2. Do NOT touch wiring or panels
3. Check for damaged or melted cables
4. Inspect junction boxes for burn marks
5. Call a certified electrician NOW
🔌 Open circuit
MODERATE
1. Check all MC4 connectors and plugs
2. Check 8A series fuse — replace if blown
3. Look for broken or disconnected wires
4. Test each panel with a multimeter
5. Call installer if issue persists
🌤 Partial shading
LOW
1. Clean panels with water and soft cloth
2. Check for dust or bird droppings
3. Check for shadow from trees or buildings
4. Trim any branches blocking sunlight
5. Schedule inspection if problem persists
⚠ Ground fault
URGENT
1. Switch OFF inverter immediately
2. Isolate PV array from inverter
3. Do NOT touch panels or wiring
4. Check insulation on all cables
5. Call certified electrician URGENTLY
🔧 Inverter fault
MODERATE
1. Check inverter display for error code
2. Switch inverter OFF and wait 30 seconds
3. Switch inverter back ON
4. Check DC and AC connections
5. Call installer if fault persists
